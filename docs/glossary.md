# Glossary of BMW OEM Tools Terminology

This glossary provides definitions for key acronyms and terms used throughout the documentation.

| Acronym | English Term | Polish Term | Description |
| :--- | :--- | :--- | :--- |
| **AOS** | **Aftersales Online System** | System Online Obsługi Posprzedażnej | Oficjalny portal BMW dla niezależnych warsztatów, oferujący dostęp do dokumentacji technicznej, instrukcji napraw i informacji o oprogramowaniu. |
| **AIR** | **Aftersales Information Research** | Badanie Informacji Posprzedażnych | Wewnętrzny system BMW do zarządzania i udostępniania informacji technicznych. |
| **CAN** | **Controller Area Network** | Sieć Kontrolerów | Standard magistrali komunikacyjnej, używany do komunikacji między modułami sterującymi (ECU) w pojeździe. |
| **D-CAN** | **Diagnostic CAN** | Diagnostyczny CAN | Wariant magistrali CAN używany do diagnostyki w starszych modelach BMW (seria E). |
| **DTC** | **Diagnostic Trouble Code** | Diagnostyczny Kod Usterki | Kod błędu generowany przez moduł sterujący (ECU) w przypadku wykrycia usterki. |
| **ECU** | **Electronic Control Unit** | Elektroniczna Jednostka Sterująca | Moduł elektroniczny sterujący jedną lub kilkoma funkcjami elektrycznymi w pojeździe (np. silnikiem, skrzynią biegów). |
| **ENET** | **Ethernet** | Ethernet | Standard komunikacji sieciowej używany do diagnostyki i programowania w nowszych modelach BMW (seria F, G, I). |
| **ICOM** | **Integrated Communication Optical Module** | Zintegrowany Moduł Komunikacji Optycznej | Oficjalny interfejs diagnostyczny BMW, używany do połączenia pojazdu z systemami ISTA/P i ISTA/D. |
| **ISTA/D** | **Integrated Service Technical Application / Diagnostics** | Zintegrowana Aplikacja Serwisowa / Diagnostyka | Oprogramowanie diagnostyczne BMW, używane do odczytu błędów, testów komponentów i procedur serwisowych. |
| **ISTA/P** | **Integrated Service Technical Application / Programming** | Zintegrowana Aplikacja Serwisowa / Programowanie | Oprogramowanie BMW do programowania, kodowania i aktualizacji modułów sterujących (ECU). |
| **K-Line** | **K-Line** | Linia K | Starszy protokół komunikacyjny (ISO 9141) używany w modelach BMW serii E do diagnostyki. |
| **KSD2** | **Kosten- und Zeitdaten** | Dane Kosztów i Czasu | System BMW do określania standardowych czasów pracy i kosztów napraw. |
| **MOST** | **Media Oriented Systems Transport** | Transport Systemów Zorientowanych na Media | Szybka magistrala optyczna używana do komunikacji między modułami multimedialnymi (np. iDrive, wzmacniacz audio). |
| **TCU** | **Transmission Control Unit** | Jednostka Sterująca Skrzynią Biegów | Moduł sterujący pracą automatycznej skrzyni biegów. |
| **VIN** | **Vehicle Identification Number** | Numer Identyfikacyjny Pojazdu | Unikalny numer identyfikujący pojazd. |
| **ZGW** | **Zentrales Gateway** | Centralna Bramka | Moduł pełniący funkcję centralnej bramki komunikacyjnej między różnymi magistralami w pojeździe. |
