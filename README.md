## Scope and Limitations

This documentation covers the following official BMW OEM software and hardware:

*   **Diagnostics & Service:** ISTA/D, INPA, Tool32, NCS Expert, WinKFP, KSD2.
*   **Programming & Coding:** ISTA/P, E-SYS.
*   **Ecosystem & Access:** ISPI Next, AOS (Aftersales Online System), AIR (Aftersales Information Research).
*   **Hardware:** ICOM (Integrated Communication Optical Module).

**What this repository DOES NOT contain:**

*   **Binary Files:** No firmware, software installers, or executable files.
*   **Pirated Sources:** No links or references to unauthorized or illegal software distribution channels.
*   **Proprietary Data:** No confidential or unreleased BMW AG internal information.

## How to Use This Repository

1.  **Select your language** using the links above.
2.  The main language-specific file (`docs/read-en.md` or `docs/read-pl.md`) provides an **overview** of all tools.
3.  For **mega-detailed** information on a specific tool, follow the "Read More" links to the corresponding file in the `docs/tools/` directory.
4.  Consult the `docs/glossary.md` for definitions of technical acronyms (**DTC**, **ECU**, **ENET**, etc.).
5.  Refer to `docs/references.md` for all cited official BMW sources.
