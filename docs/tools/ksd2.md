# KSD2 (Kosten- und Zeitdaten)

## 1. Title and Short Description

**KSD2** (Kosten- und Zeitdaten, or Cost and Time Data) is the official BMW Group application used to determine the standard labor times and costs for service and repair operations. It is an essential tool for service advisors and technicians for accurate job quoting and warranty claim processing.

## 2. Purpose / Use Cases

KSD2 is used for administrative and financial aspects of the service process.

*   **Job Quoting:** Generating accurate estimates for repair and maintenance work based on standardized labor times.
*   **Warranty Claims:** Providing the necessary labor codes and times for submitting warranty claims to BMW AG.
*   **Service Planning:** Assisting in scheduling and planning service appointments based on the estimated duration of the work.

## 3. Supported Vehicle Generations

KSD2 contains data for all current and past BMW Group vehicles.

## 4. Interfaces & Protocols

KSD2 is typically a standalone application or integrated into the dealer management system (DMS).

*   **Network Access:** Requires a network connection for data updates and synchronization.

## 5. Installation & Access

KSD2 is usually installed as part of the dealer management system.

*   **Legal Access:** Access is restricted to authorized BMW Group dealers and service partners.

## 6. General Workflow / Typical Procedures

1.  **Vehicle Identification:** Input the vehicle's **VIN** (Vehicle Identification Number).
2.  **Select Repair:** Select the specific repair or service operation (e.g., "Replace brake pads front").
3.  **Retrieve Data:** KSD2 retrieves the official labor time (in work units or minutes) and the corresponding labor code.
4.  **Quoting:** The data is used to generate a customer quote or a warranty claim.

## 7. Key Screens / UI Elements

*   **Labor Code Search:** Interface for searching for specific repair operations.
*   **Time Unit Display:** Shows the standard time required for the selected job.

## 8. Safety & Risks

The main risk is using outdated data, which can lead to incorrect quotes or rejected warranty claims.

*   **Data Accuracy:** Technicians must ensure the KSD2 database is up-to-date with the latest data from BMW AG.

## 9. Troubleshooting & Common Errors

| Error | Description | Diagnosis / Solution |
| :--- | :--- | :--- |
| **Data Mismatch** | KSD2 data does not match the latest official data. | Synchronize the KSD2 database with the BMW backend. |

## 10. Examples & Practical Notes

*   **Labor Codes:** KSD2 provides the specific labor codes required for every operation, which are essential for internal tracking and external billing.

KSD2 data is updated frequently to reflect changes in repair procedures and new vehicle models.

## 13. Role

KSD2 is used by **Service Advisors** and **Warranty Administrators** to manage the financial and time aspects of service operations.
